# User Prompt
Also on the right side top menu, give a new menu item with label 'Call Back'.
Here the idea is to simulate an incoming call.
Show an Incoming call screen, so that user can accept the call and then that particular contact's audio starts playing.
When The user sets the call back, he should have to option to select the Contact, as well as the time delay after which the incling call is triggered

# Implementation Plan - Add Call Back (Incoming Call Simulator)

We will implement a "Call Back" configuration screen to simulate scheduled incoming calls. When the callback fires, it displays a realistic Incoming Call screen playing the ringtone. Accepting transitions to `InCallActivity`.
